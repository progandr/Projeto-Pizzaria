🍕 Site de Pizzaria

Projeto de um site de pizzaria desenvolvido com foco em apresentação de produtos e geração de pedidos, com layout moderno, responsivo e integração para contato rápido.

🚀 Funcionalidades
Exibição do cardápio de pizzas
Layout responsivo (mobile e desktop)
Botão de contato via WhatsApp
Navegação simples e intuitiva
Destaque para promoções e produtos
🛠️ Tecnologias utilizadas
HTML5
CSS3
JavaScript
📱 Objetivo

Criar um site funcional e atrativo para pizzarias, facilitando a visualização do cardápio e incentivando o cliente a realizar pedidos de forma rápida.
